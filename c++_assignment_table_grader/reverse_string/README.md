# #2 Reverse String (C++)

You are given 1 string as argument. Print the reversed string.
Solution should contain 1 `Makefile` and any `.h` and `.cpp` files.
Running `make` should produce the executable `reverse`.

### Examples
* jam -> maj
* horse -> esroh
* desserts -> stressed
* flow -> wolf
