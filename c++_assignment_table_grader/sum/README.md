# Sum (C++)

[< Home](../../README.md)

You are given many integers as arguments. Print the sum of all integers.
Solution should contain 1 `Makefile` and any `.h` and `.cpp` files.
Running `make` should produce the executable `sum`.

### Examples

```bash
./sum 2 2 # 4
./sum 3 6 # 9
./sum 42 16 32 # 90
./sum 3 6 9 12 15 # 45
```
