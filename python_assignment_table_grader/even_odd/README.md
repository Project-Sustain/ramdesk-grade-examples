# Even Odd (Python)

[< Home](../../README.md)

You are given many integers as arguments. Print "even" or "odd" for each integer.
Solution should contain 1 file `even_odd.py`.

### Examples

```bash
python even_odd.py 2 2 # even even
python even_odd.py 3 6 # odd even
python even_odd.py 17 36 722 # odd even even
python even_odd.py 3 6 9 12 15 762948 3948535 # odd even odd even odd even odd
```
